# AI Resume Analyzer + Job Fit Predictor

## Overview

This project is an AI-powered Resume Analyzer and Job Fit Predictor built using **Endee Vector Database**.  
It evaluates a candidate’s resume against a job description using **semantic embeddings**, providing insights into:

- Matching skills
- Missing skills
- Job fit score
- Suggestions for improvement

The system demonstrates a real-world AI/ML workflow using **vector databases**, **semantic search**, and **RAG (Retrieval Augmented Generation)**.

---

## Features

1. **Resume Upload:** Upload your PDF resume.
2. **Job Description Input:** Enter job description text.
3. **Skill Extraction:** Automatically extracts skills from resume and job description.
4. **Semantic Matching:** Computes similarity using embeddings stored in Endee.
5. **Results Display:** Shows:
   - Skills matched
   - Missing skills
   - Job fit score (%)
   - Optional improvement suggestions
6. **Endee Integration:** Stores embeddings in Endee vector database for fast, scalable semantic search.
